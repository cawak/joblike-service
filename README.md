<h1>JobLike</h2>


## Description

Job like is an application for job seekers for mobile devices with the idea of swipping just like with Tinder app, but this time, you get only matches!

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
